### Install
To install the dependencies, run:
 ~~~
npm install
 ~~~

### Run on the localhost
Run the command:
 ~~~
npm start
 ~~~

### Dependencies:
  - nodejs:
    - socket.io
    - express
  - jquery
  - boostrap
